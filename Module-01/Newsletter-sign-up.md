# Newsletter Sign-Up Form with Success Message

## Project Objectives

- Use HTML to structure the form and CSS to style it, ensuring a consistent and accessible design across different browsers.
- Implement JavaScript for real-time form validation, showing error and success messages dynamically based on user input.
- Apply responsive design techniques with CSS media queries to make the form adaptable to various screen sizes and enhance user interaction with hover and focus states.

---

## Project Description

Your task is to build out this newsletter form and get it looking as close to the design as possible.

### Your users should be able to:

- Add their email and submit the form
- See a success message with their email after successfully submitting the form
- See form validation messages if:
  - The field is left empty
  - The email address is not formatted correctly
- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page
